App Structure

All the common areas of app like header, navigation menu, settings panel, footer, fixed footer menu etc. are included from a common file under Our App-phonegap/www/includes folder. This files are loaded using jQuery .load() method.

This approach makes our code easy to manage.

App Configuration

All options available in variables.js file

All the settings related to app can be found in Our App-phonegap/www/assets/js/variables.js

You can set App default theme color, menu style, menu color, site mode etc. in this file.

var defaultval = {
  theme: 'deep-purple',
  header: 'light',
  header_align: 'center',
  menu: 'light',
  menu_icons: 'on',
  menu_type: 'left',
  footer: 'light',
  footer_type: 'minimal',
  site_mode: 'light',
  footer_menu: 'hide',
  footer_menu_style: 'light',
}

Also you can set time after which user login expires and user must reenter password to continue using this app features. Update this value var AUTHEXPIREMILLISECONDS = 86400000.

var restricted = [] array helps in managing list of pages that require user to login for access.

Other App language related settings are stored and can be added in this file.

Framework

Our App: Multi Purpose Mobile App is built using the famous CSS Framework "Materialize" (https://materializecss.com/). Materialize is a modern responsive front-end framework based on Material Design.

Created and designed by Google, Material Design is a design language that combines the classic principles of successful design along with innovation and technology. Google's goal is to develop a system of design that allows for a unified user experience across all their products on any platform.

Offsets

To offset, simply add offset-s2 to the class where s signifies the screen class-prefix (s = small, m = medium, l = large) and the number after is the number of columns you want to offset by.


    <div class="row">
      <div class="col s12"><span class="flow-text">This div is 12-columns wide on all screen sizes</span></div>
      <div class="col s6 offset-s6"><span class="flow-text">6-columns (offset-by-6)</span></div>
    </div>

Push and Pull

You can easily change the order of your columns with push and pull. Simply add push-s2 or pull-s2 to the class where s signifies the screen class-prefix (s = small, m = medium, l = large) and the number after is the number of columns you want to push or pull by.


    <div class="row">
      <div class="col s7 push-s5"><span class="flow-text">This div is 7-columns wide on pushed to the right by 5-columns.</span></div>
      <div class="col s5 pull-s7"><span class="flow-text">5-columns wide pulled to the left by 7-columns.</span></div>
    </div>

Site Mode

"Our App: Multi Purpose Mobile App" can be viewed as light and dark mode.

You can keep both the versions in your project or show only as light or dark.

There is a icon in header to switch between light and dark mode.

   

Open <body> tag has attribute as data-site_mode="dark" for dark mode. Light is default mode. You can also mention data-site_mode="light" for light mode in your app. The switch icon in header just toggles this attribute value to light and dark when clicked.

Settings Menu

Settings Menu contains all the settings of your app template. User can configure their own settings. Settings menu has options for themes, header layout, header style, footer layout and style, menu layout and style, Fixed Bottom Menu, Menu Icons, Site mode etc.

Fixed Bottom Menu

Fixed Bottom Menu is the short cut on bottom of every page in this template. It is designed to have maximum 5 links as per your need. It is configured to have light, dark and colored style.

Fixed Bottom Menu style can be applied from settings side bar of this template.

<body> tag has attribute data-footer_menu="show" to display Fixed Bottom Menu.
<body> tag has attribute data-footer_menu="hide" to hide Fixed Bottom Menu.


<body> tag has attribute data-footer_menu_style="light" for Light Colored Fixed Bottom Menu style.

<body> tag has attribute data-footer_menu_style="dark" for Dark Colored Fixed Bottom Menu style.

<body> tag has attribute data-footer_menu_style="colored" for Colored Fixed Bottom Menu style. Color is based on your theme used.

assets/js/chart-echarts.js has all the code implemented for each graph type. You can visit the Charts section of this template for further reference.
